# 雪中悍刀行 后台项目

## 如何安装

```bash
# 安装依赖
$ npm install

# 在localhost:3000上启动项目
$ npm run dev

# 打包项目用于部署到服务器上
$ npm run build
$ npm start

```
## 介绍
![](./static/20200916002303.png)
